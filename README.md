# Electric-Vehicle-PowerBI-Dashboard
Power BI dashboard analyzing 150K+ electric vehicles across the US


## Key Metrics
- **Total Vehicles:** 150.42K
- **Avg EV Range:** 67.83 Miles
- **BEV Vehicles:** 117K (77.61%)
- **PHEV Vehicles:** 34K (22.39%)

## Visuals Included
- Total Vehicles by Model Year
- Total Vehicles by State (US Map)
- Top Makes: Tesla, Nissan, Chevrolet, Ford
- CAFV Eligibility Breakdown
- Vehicle Model Matrix
